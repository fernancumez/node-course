<!-- GETTING STARTED -->

## Getting started

1. Installation

- Clone the repo

```
git clone https://github.com/fernancumez/node-review.git
```

2. cd into folder

```
cd node-review
```

3. Download dependencies

```
npm install
```

<!-- USAGE -->

## Usage

- Run `npm run watch` for compilation mode
- Run `npm run files`
- Run `npm run classes`
- Run `npm run promises`
